# music-player

>   移动端音乐播放器,整合网易云音乐与QQ音乐,避免听歌时来回切换音乐播放器的窘境

[点此预览](http://music.dingjian.name)

## 功能预览

![](http://image.dingjian.name/blog/170920/D2k8m615G6.jpg?imageslim)


## 运行项目

``` bash
# 克隆到本地
git clone 

# 安装项目依赖
npm install

# 运行
npm run dev

# 打包构建
npm run build
```
